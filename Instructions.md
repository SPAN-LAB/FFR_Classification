# How to use:

1. Clone the branch `anu_dev` from the repo
2. Create a new virtual environment and activate it:
    -`python3 -m venv .venv`
    -`source .venv/bin/activate`
3. `pip install -r requirements.txt`
    -if the installation fails for any, please manually install them
4. Copy pdemo.py into this directory
5. Ensure you're connected to the research drive
6. Note: the file paths in pdemo.py are only compatible with Macs,
    if you are running it on Windows, some adjustments to the paths might be necessary. 
7. Run pdemo.py: `python pdemo.py`
